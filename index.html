Owner Portal

A Netlify-ready HTML front door for the full Ancestral Healing repository. Search every shipped file, preview dashboards and PDFs, and read markdown documents without leaving the browser.

Ready for Netlify drag-and-drop

The deployment zip keeps the repo structure intact and places this index.html at the root, so Netlify can serve it immediately with no build step.

  1. Open Netlify and choose Add new site → Deploy manually.
  2. Drop ancestral-healing-netlify-drop.zip into the upload area.
  3. Open the generated Netlify URL and use this portal as the owner's home base.
